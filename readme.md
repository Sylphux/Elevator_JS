# About

This repo contains a working html / css / js page that allows you to play with an interactive elevator.

The elevator can be called at multiple levels and it will stop depending on if it has to or not.

Access the page here : https://sylphux.github.io/Elevator_JS/ 

## Instructions

The exercice was given to me in this form : 

"Tu doit coder un ascenseur fonctionnel. Tu peut faire sa avec un visuelle ou en console. Le but est que l'ascenseur se comporte de manière logique avec les usagers. SI qlq appel l'ascenseur, il se rend l'étage. Mais si deux personnes l'appellent, alors l'ascenseur donne la priorité à ceux qui montent. L'ascenseur ignore tous les appels lors qu'il monte un usager à l'étage choisie mais lors ce qu'il descend il récupère tous le monde. ect..."

## How it works

I've decided to create the visuals with html / css and to move the divs with JS using absolute / fixed positions for the elevator.

The game console logs any actions taken by the player and gives other infos on what is happening in the program. It replaces the javascript console for developing and learning purposes.
